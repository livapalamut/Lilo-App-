# Lilo: Teknoloji Yığını

Bu projede hız ve etkinlik için aşağıdaki modern araçlar seçilmiştir:

* [cite_start]**Frontend:** React ve Tailwind CSS (Hızlı UI geliştirme ve modern tasarım için). [cite: 40]
* [cite_start]**AI Model:** Gemini Pro (Google AI Studio üzerinden API ile bağlanarak duygu analizi ve rasyonel sorgulama için). [cite: 40]
* [cite_start]**Kod Editörü:** Cursor AI (Agent modu ile "Vibe Coding" yaparak hızlı prototipleme için). [cite: 40]
* [cite_start]**Versiyon Kontrol:** GitHub (Kodların saklanması ve teslimi için). [cite: 13, 25]
* [cite_start]**Yayınlama (Deploy):** Lovable (Saniyeler içinde canlıya almak için). [cite: 40]
